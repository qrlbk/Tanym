# Разработка

Краткий обзор стека и структуры репозитория. Подробная установка — в [README.md](../README.md).

## Позиционирование

**Tanym** — AI-редактор для авторов романов и длинной прозы на русском. Продуктовое ядро — writer-copilot, который знает проект целиком (`StoryProject → Chapter → Scene → Character`) и умеет делать cross-scene правки и проверку continuity.

При любом спорном решении (добавить фичу / оставить / выкинуть) ориентир — **одиночный автор романа**, а не офисный документооборот. Mailings / академические References / шаблоны писем и договоров сознательно исключены.

## Стек

| Слой | Технологии |
|------|------------|
| UI | React 19, Next.js 16, Tailwind CSS 4 |
| Редактор | TipTap 3, ProseMirror (`@tiptap/pm`) |
| Состояние | Zustand |
| Десктоп | Tauri 2 (`src-tauri/`) |
| Тесты | Vitest |

## Структура каталогов (кратко)

| Путь | Назначение |
|------|------------|
| `src/app/` | Маршруты и layout Next.js (App Router) |
| `src/components/` | Компоненты UI, лента, редактор, диалоги |
| `src/components/Editor/` | Провайдер редактора, расширения TipTap |
| `src/lib/` | Утилиты, ИИ-клиент, работа с файлами |
| `src/stores/` | Zustand-хранилища |
| `src-tauri/` | Rust-код и конфигурация Tauri |
| `scripts/` | Скрипты обслуживания (`setup.mjs` и др.) |

## Команды

| Команда | Назначение |
|---------|------------|
| `npm run dev` | Разработка веб-версии |
| `npm run build` | Сборка Next.js |
| `npm run tauri:dev` | Разработка с окном Tauri |
| `npm run tauri:build` | Сборка десктопного приложения |
| `npm run lint` | ESLint |
| `npm run test` | Запуск Vitest |

## Next.js

В проекте используется сборка Next.js с особенностями текущей мажорной версии.
Перед изменениями в маршрутизации, конфиге или API имеет смысл свериться с
`node_modules/next/dist/docs/` в установленном пакете.
