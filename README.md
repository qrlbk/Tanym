# Word.kz

Word-подобный редактор на Next.js (TipTap): страницы с полями, импорт/экспорт DOCX, опционально десктоп через Tauri.

## Требования

- Node.js 20+
- npm

## Установка

```bash
npm install
```

## Переменные окружения

Скопируйте пример и вставьте ключ API:

```bash
cp .env.example .env.local
```

Заполните `OPENAI_API_KEY` для функций ИИ.

## Скрипты

| Команда | Описание |
|--------|----------|
| `npm run dev` | Режим разработки |
| `npm run build` | Сборка продакшена |
| `npm run start` | Запуск после сборки |
| `npm run lint` | ESLint |
| `npm run tauri:dev` | Десктоп (Tauri) в dev |
| `npm run tauri:build` | Сборка десктоп-приложения |
