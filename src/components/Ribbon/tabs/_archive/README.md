# Архив ribbon-вкладок

Вкладки, выключенные после смены позиционирования продукта на writer-copilot
для романистов. Не импортируются из панели инструментов; сохранены как
референс, чтобы при необходимости вернуть или скопировать части UI.

- `MailingsTab.tsx` — рассылки (почтовые слияния). Писателю романов не нужны.
- `ReferencesTab.tsx` — академические ссылки и библиография. То же.
- `ReviewTab.tsx` — рецензирование / правописание (перенесено во вкладку «Вид»).
- `LayoutTab.tsx` — макет страницы (скрыт из writer-UX; геометрия docPage сохранена).

См. `docs/DEVELOPMENT.md` → «Позиционирование».
